## 3. CPU Information — /proc/cpuinfo

### Command
cat /proc/cpuinfo | head -30

### What It Shows
Displays information about the system's CPU from the Linux `/proc` virtual filesystem.

It includes details such as:
- CPU model
- processor number
- CPU cores
- cache size
- CPU frequency
- CPU features and flags

The command `head -30` limits the output to the first 30 lines.

### Observations
- The file lists processor information for the system CPU.
- It includes fields like:
  - processor
  - vendor_id
  - model name
  - cpu MHz
  - cache size
  - flags
- Each processor section represents a CPU core or logical processor.
